# Secure Voting System - SVV Project

This project implements a Secure Voting System as part of the Software Verification and Validation (SVV) Lab.

## Objective
The goal is to design and formally verify a voting system that ensures:
- Integrity (no duplicate voting)
- Anonymity (no voter-vote linkage)
- Security (only authorized voters can vote)

## Key Features
- Voter registration and authentication
- Token-based voting mechanism
- Anonymous vote storage
- Prevention of multiple voting
- Accurate vote counting

## Technologies Used
- Z Notation (Formal Modeling)
- VDM (Formal Specification)
- Alloy Analyzer (Verification)
- GitHub (Version Control & Issue Tracking)

## Project Structure
- `/requirements` → SRS and requirement analysis  
- `/z-model` → Z formal model  
- `/vdm-spec` → VDM specification  
- `/alloy-model` → Alloy verification  
- `/validation` → validation and security  
- `/ci-pipeline` → CI/CD configuration
